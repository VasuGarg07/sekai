# 🌏 Sekai Arcadia

Sekai Arcadia is a modern anime web application that combines **tracking, watchlisting, and catalogue browsing** into a seamless user experience.
Built with a sleek UI and powerful backend, it helps users discover, organize, and manage their anime journey.

---

## ✨ Features

* 📺 **Anime Catalogue Browser**
  Explore trending, popular, top airing, and completed anime.

* ⭐ **Watchlist Management**
  Add anime to your personal watchlist with a single click.

* 🔍 **Search & Filter**
  Quickly find anime using a responsive search system.

* 🎯 **Spotlight Section**
  Highlights featured anime with detailed previews.

* ⚡ **Fast & Responsive UI**
  Optimized for smooth performance across devices.

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/sekai-arcadia.git

# Navigate into the project
cd sekai-arcadia

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add your Firebase config:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
